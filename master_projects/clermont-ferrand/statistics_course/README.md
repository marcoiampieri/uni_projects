### This folder contains the Jupiter Notebook submitted as part of the final grade for the _Statistics_ module of the _Statistics and data analysis_ course in Clermont-Ferrand, during the $1^{st}$ semester of my Master's programme.

## Task

For this project, we were asked to test statistical hypotheses for the digit frequencies of different notorious numbers.

1) Test the _Normal Number_ hypothesis (i.e. a number that has an equal frequency for each of its digits) for the following numbers:
   * **$\pi$**
   * **$e$** (Euler's number)
   * **$\sqrt{2}$**
   * **$\phi$** (Golden Ratio)
   * **Fibonacci series**
   * **Lucas sequence**
   * **Padovan sequence**

2) Test the _law of anomalous numbers_ using the _Newcomb-Benford_ probability mass function for the following cases:
   * **First digit of the Fibonacci numbers**
   * **Multiplication of randomly independent variables**
   * **$Z={(\frac{X}{Y})}^n$** where both X and Y are normally distributed
